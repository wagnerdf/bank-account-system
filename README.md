# Bank Account System

Sistema de cadastro de usuários e contas bancárias, com:
- Backend em Spring Boot
- Frontend em React
- Integrações REST/JSON
- Integração SOAP/XML para cálculo de impostos

## Estrutura do projeto

- backend/ → API Spring Boot
- frontend/ → Aplicação React
- docs/ → Documentação (MER, arquitetura, diagramas)

## Perfis de usuário
- USER
- ADMIN
